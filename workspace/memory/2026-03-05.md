## ACP (Agent Client Protocol) — OpenClaw
- ACP = протокол подключения IDE (Zed и др.) к OpenClaw Gateway через stdio
- acpx = кастомное расширение с dispatch (маршрутизация между агентами) и permissionMode
- allowedAgents в acpx: pi, claude, codex, opencode, gemini, kimi
- Используется для мультиагентных шлюзов — одна точка входа, выбор модели внутри
