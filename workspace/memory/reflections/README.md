# Reflections Index

Дневник рефлексий KZ. Каждый файл — отдельная тема.
Формат внутри: датированные записи (новые сверху).
Сам текст записей может быть на русском.

## Topics
- `natasha.md` — отношения с Наташей
- `health.md` — здоровье, самочувствие, привычки
- `business.md` — бизнес, проекты, решения
- `personal.md` — личное, мировоззрение, цели

## How to search
- memory_search: ищи по теме → найдёт упоминание в MEMORY.md
- memory_read: читай `memory/reflections/<topic>.md` напрямую
