## Изоляция агентов в OpenClaw
- KZ хочет создать публичного бота для любых людей с отдельными инструкциями
- Обсудили варианты: отдельный бот + отдельный агент + sandbox
- sandbox mode:"all" + tools.deny:["exec"] = двойная защита, реальная техническая изоляция
- Без sandbox: read ограничен workspace, но exec может выйти за пределы
